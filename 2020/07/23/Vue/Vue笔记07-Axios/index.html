介绍

Axios是一个基于promise的HTTP库，适用于浏览器和node.js。

特点

  • 浏览器端发起XMLHttpRequests请求
  • node端发起http请求
  • 支持Promise API
  • 拦截请求和返回
  • 转化请求和返回（数据）
  • 取消请求
  • 自动转化json数据
  • 客户端支持抵御XSRF（跨站请求伪造）

拦截器

请求拦截的作用：

  • config中一些信息不符合服务器的需求。
  • 发送网络请求时希望在界面上显示一个请求的图标。
  • 一些网络请求(如登录)，必须携带一些特殊信息。

响应拦截的作用：

  • 对请求的数据进行过滤。
  • 根据错误码跳转到不同的提示页面。
